🐺 Get Started in 5 Minutes

Follow these steps while you wait for your license email.

  1. Install the packages
    npm install @ergenekon/probe @ergenekon/collector
  2. Save your license file
    Download .ergenekon-license.json from your email and place it in your project root directory.
  3. Add the probe to your Express app
    import { ErgenekonProbe } from '@ergenekon/probe';
    const probe = new ErgenekonProbe({ serviceName: 'my-api' });
    app.use(probe.expressMiddleware());
  4. Start the collector & dashboard
    npx @ergenekon/collector --port 4380
    npx @ergenekon/ui          # → localhost:3001
  5. Make requests and inspect
    Open http://localhost:3001 — your sessions appear in real-time. Click Play to step through every I/O event. 🎉
